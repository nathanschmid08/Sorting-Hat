Einführung in HTML

HTML (Hypertext Markup Language) ist eine Auszeichnungssprache zur Strukturierung von Webinhalten.

Beispiel


<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Meine Webseite</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Willkommen auf meiner Webseite</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#section1">Section 1</a></li>
            <li><a href="#section2">Section 2</a></li>
        </ul>
    </nav>
    <section id="section1">
        <p>Dies ist der Inhalt von Section 1.</p>
    </section>
    <section id="section2">
        <p>Dies ist der Inhalt von Section 2.</p>
    </section>
    <footer>
        <p>© 2023 Meine Webseite</p>
    </footer>
</body>
</html>
