Einführung

TypeScript ist eine moderne Programmiersprache, die auf JavaScript basiert und dieses um zusätzliche Funktionen erweitert. Der wichtigste Unterschied: TypeScript erlaubt die Verwendung von sogenannten Typen, mit denen du genauer festlegen kannst, welche Art von Daten dein Code erwartet. Das hilft dir, Fehler frühzeitig zu erkennen und macht deinen Code übersichtlicher und besser wartbar – besonders in größeren Projekten. TypeScript wird vor der Ausführung in normales JavaScript umgewandelt, sodass es in jedem Browser funktioniert. Es ist heute aus der Webentwicklung kaum mehr wegzudenken, vor allem wenn Teams an komplexeren Anwendungen arbeiten.

Beispiel


function greet(name: string): string {
    return "Hallo, " + name + "!";
}
              
let userName: string = "Nathan";
              
console.log(greet(userName));
