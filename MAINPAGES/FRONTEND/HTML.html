Einführung

HTML (Hypertext Markup Language) ist die grundlegende Auszeichnungssprache des Webs. Sie wird verwendet, um den Aufbau und die Struktur von Inhalten auf Webseiten festzulegen. Mit HTML kannst du bestimmen, wo Texte, Überschriften, Bilder, Links, Listen und viele andere Elemente auf einer Seite angezeigt werden sollen. Dabei sagt HTML dem Browser nicht, wie etwas aussehen soll (das übernimmt CSS), sondern was für Inhalt es ist und welche Bedeutung er hat. Ohne HTML gäbe es keine lesbare oder funktionierende Webseite – es ist also das Grundgerüst jeder Internetseite.

Beispiel


<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Meine Webseite</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Willkommen auf meiner Webseite</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#section1">Section 1</a></li>
            <li><a href="#section2">Section 2</a></li>
        </ul>
    </nav>
    <section id="section1">
        <p>Dies ist der Inhalt von Section 1.</p>
    </section>
    <section id="section2">
        <p>Dies ist der Inhalt von Section 2.</p>
    </section>
    <footer>
        <p>© 2023 Meine Webseite</p>
    </footer>
</body>
</html>



<!DOCTYPE html>

Die <!DOCTYPE html> Deklaration steht ganz am Anfang jeder HTML-Datei und teilt dem Browser mit, welche Version von HTML verwendet wird. In diesem Fall zeigt <!DOCTYPE html> an, dass es sich um HTML5, die aktuellste Version von HTML, handelt. Er sorgt dafür, dass der Browser die Webseite im sogenannten „Standardmodus“ rendert – also ohne alte Kompatibilitätsfehler. Ohne diesen Tag könnten Webseiten anders oder fehlerhaft angezeigt werden, besonders in älteren Browsern.




<html>

Der <html> Tag ist das äusserste Element jeder HTML-Seite. Er sagt dem Browser: „Alles, was hier drin steht, gehört zur HTML-Seite.“

Beispiel:


<!DOCTYPE html>
<html>

    <!--Hier kommt der Inhalt-->

</html>



<head>

Der <head> Tag steht im oberen Teil deiner HTML-Seite und enthält Informationen über die Webseite, die nicht direkt im Browser sichtbar sind. Man nennt diesen Bereich auch „Kopf der Seite“. Hier kommen z. B. der Seitentitel, Zeichencodierung, Verlinkungen zu CSS-Dateien, Meta-Informationen und mehr hinein.

Beispiel:


<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Meine Webseite</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!--Hier kommt der Inhalt-->
</body>
</html>
