Einführung

CSS (Cascading Style Sheets) ist die Sprache, mit der das Aussehen einer Webseite gestaltet wird. Während HTML den strukturellen Aufbau einer Seite definiert, bestimmt CSS, wie diese Inhalte dargestellt werden – zum Beispiel Farben, Abstände, Schriftarten oder Layouts. Mit CSS kannst du eine Webseite modern, einheitlich und ansprechend gestalten, ohne den HTML-Code unübersichtlich zu machen. Es trennt also das Design vom Inhalt und sorgt dafür, dass Webseiten nicht nur funktionieren, sondern auch gut aussehen.

Beispiel


<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>CSS-Beispiel</title>
    <style>
        /* Das ist CSS */
        h1 {
            color: #4186C2;                     /* Textfarbe */
            text-align: center;                 /* Zentriert den Text */
            font-family: Arial, sans-serif;     /* Schriftart */
        }

        p {
            color: #333;
            font-size: 16px;
            line-height: 1.5;
        }
    </style>
</head>
<body>

    <h1>Hallo CSS!</h1>
    <p>Mit CSS kannst du das Aussehen deiner Webseite gestalten.</p>

</body>
</html>



Selektoren

CSS-Selektoren sind Muster, mit denen du gezielt HTML-Elemente auswählst und gestaltest. Sie bestimmen, auf welche Elemente deine CSS-Regeln angewendet werden – zum Beispiel über Tag-Namen, Klassen oder IDs. Mit komplexeren Selektoren kannst du sogar Elemente nach ihrer Position im DOM oder nach Attributen ansprechen. Durch präzise Selektoren behältst du die Kontrolle über das Design und vermeidest unerwünschte Überschneidungen.

Beispiel


/* Element-Selektor: alle <p> Elemente */
p {
    color: blue;
}
            
/* Klassen-Selektor: alle Elemente mit class="highlight" */
.highlight {
    background-color: yellow;
}
            
/* ID-Selektor: nur das Element mit id="spezial" */
#spezial {
    font-weight: bold;
}



Deklarationen

CSS-Deklarationen legen fest, wie ein ausgewähltes HTML-Element dargestellt werden soll. Jede Deklaration besteht aus einer Eigenschaft (z. B. color, font-size oder margin) und einem dazugehörigen Wert. Mehrere Deklarationen werden in geschweifte Klammern geschrieben und durch Semikolons getrennt – sie bilden zusammen einen sogenannten Deklarationsblock. Mit diesen Deklarationen kannst du das Aussehen deiner Webseite gezielt beeinflussen, von Farben über Abstände bis hin zu Animationen.

Beispiel


/* Grundlegendes Styling für die gesamte Seite */
body {
    background-color: #f0f0f0;     /* Hellgrauer Hintergrund für die Seite */
    font-family: sans-serif;       /* Serifenlose Schriftart für besseren Lesefluss */
    margin: 0;                     /* Entfernt den Standard-Aussenabstand des Browsers */
    padding: 20px;                 /* Fügt innen etwas Abstand zum Rand hinzu */
}
            
/* Überschrift 1 (z. B. Seitentitel) */
h1 {
    color: #2c3e50;                /* Dunkelblauer Farbton für den Text */
    font-size: 32px;               /* Grosse Schriftgrösse */
    margin-bottom: 10px;           /* Abstand unterhalb der Überschrift */
}
            
/* Fliesstext (Absätze) */
p {
    color: #555555;                /* Mittelgraue Textfarbe */
    font-size: 16px;               /* Normale Schriftgrösse für Text */
    line-height: 1.6;              /* Zeilenhöhe für bessere Lesbarkeit */
}
            
/* Links im Text */
a {
    color: #007acc;                /* Blau für Links */
    text-decoration: none;         /* Entfernt die Unterstreichung */
}
            
/* Hover-Zustand für Links (wenn man mit der Maus drüberfährt) */
a:hover {
    text-decoration: underline;    /* Zeigt beim Darüberfahren eine Unterstreichung */
}



Eigennschaften und Werte

In CSS bestehen die meisten Stilregeln aus sogenannten Eigenschaften und Werten. Die Eigenschaft (englisch: property) beschreibt, was genau am Element geändert werden soll – zum Beispiel die Farbe, die Schriftgrösse oder der Abstand. Der Wert (englisch: value) legt dann fest, wie diese Eigenschaft aussehen soll – also z. B. red für die Farbe oder 20px für einen Abstand. Beides zusammen nennt man eine Deklaration und sie sieht immer so aus: eigenschaft: wert;. Mit dieser einfachen Struktur kannst du das Aussehen jeder Webseite gezielt gestalten.

Beispiel


/* Element-Selektor: Alle <body>-Elemente */
body {
    background-color: #f9f9f9;          /* Heller Hintergrund */
    font-family: Arial, sans-serif;     /* Serifenlose Standardschrift */
    padding: 40px;                      /* Innenabstand zum Rand */
}
                
/* ID-Selektor: Nur das Element mit id="haupttitel" */
#haupttitel {
    color: #2c3e50;                     /* Dunkelblauer Titeltext */
    font-size: 36px;                    /* Grosse Schrift */
    text-align: center;                 /* Zentrierter Text */
    margin-bottom: 30px;                /* Abstand zum nächsten Element */
}
                
/* Klassen-Selektor: Alle Elemente mit class="info-text" */
.info-text {
    font-size: 18px;                    /* Etwas grössere Schrift */
    color: #555555;                     /* Dunkelgrauer Fliesstext */
    line-height: 1.6;                   /* Angenehmer Zeilenabstand */
    background-color: #ffffff;          /* Weisser Hintergrund für die Textbox */
    padding: 20px;                      /* Innenabstand */
    border-left: 5px solid #007acc;     /* Farbiger Streifen am Rand */
    margin-bottom: 20px;                /* Abstand zu anderen Absätzen */
    border-radius: 4px;                 /* Leicht abgerundete Ecken */
}



Elementselektoren

Elementselektoren sind eine der grundlegendsten Methoden in CSS, um HTML-Elemente anzusprechen und zu gestalten. Dabei wählst du einfach den Namen eines HTML-Tags, zum Beispiel p für Absätze oder h1 für Überschriften, aus. Alle Elemente dieses Typs auf der Webseite erhalten dann die definierten Stile. Elementselektoren eignen sich besonders gut, um eine einheitliche Gestaltung für alle gleichen Elemente festzulegen, ohne dass du Klassen oder IDs vergeben musst. Sie sind einfach zu verstehen und bilden die Basis jeder CSS-Anwendung.

Beispiel


/* Wendet Styles auf das gesamte <body>-Element an */
body {
    background-color: #eef2f7;              /* Hellblauer Hintergrund */
    font-family: 'Segoe UI', sans-serif;    /* Moderne, gut lesbare Schrift */
    color: #333333;                         /* Dunkelgraue Standardschriftfarbe */
    margin: 40px;                           /* Aussenabstand zum Browserrand */
}
                
/* Gestaltet alle <h1>-Elemente */
h1 {
    font-size: 36px;                        /* Grosse Überschrift */
    text-align: center;                     /* Zentriert */
    color: #2a5d9f;                         /* Blauer Farbton */
    margin-bottom: 30px;                    /* Abstand nach unten */
}
                
/* Gestaltet alle <p>-Absätze */
p {
    font-size: 18px;                        /* Lesbare Textgrösse */
    line-height: 1.6;                       /* Etwas mehr Zeilenabstand */
    background-color: #ffffff;              /* Weisser Hintergrund für Lesbarkeit */
    padding: 20px;                          /* Innenabstand im Absatz */
    border-left: 5px solid #4a90e2;         /* Blauer Streifen links */
    margin-bottom: 20px;                    /* Abstand zu nächsten Absätzen */
    border-radius: 6px;                     /* Abgerundete Ecken */
}
                
/* Gestaltet alle <a>-Links */
a {
    color: #e74c3c;                         /* Roter Farbton für Links */
    text-decoration: none;                  /* Unterstreichung entfernen */
    font-weight: bold;                      /* Fettgedruckt */
}
                
a:hover {
    text-decoration: underline;             /* Unterstreichung beim Darüberfahren */
}



Klassenselektoren

Klassenselektoren ermöglichen es dir, gezielt HTML-Elemente mit einer bestimmten class-Eigenschaft zu gestalten. Du verwendest sie, indem du in CSS einen Punkt (.) vor den Klassennamen setzt – zum Beispiel .btn. So kannst du mehrere Elemente gleich stylen, ohne sie einzeln ansprechen zu müssen. Klassenselektoren sind ideal, um wiederverwendbare Gestaltungselemente wie Buttons, Karten oder spezielle Textelemente zu definieren. Sie machen deinen CSS-Code übersichtlicher und flexibler – besonders bei grösseren Webseiten.

Beispiel


index.html


<button class="button>>Klick mich>/button>
<p class="highlight">Dieser Text ist hervorgehoben.</p>

style.css


/* Definiert eine Klasse "button" für alle Buttons */
.button {
    background-color: #3498db;  /* Hintergrundfarbe blau */
    color: white;               /* Textfarbe weiss */
    padding: 10px 20px;         /* Innenabstand */
    border-radius: 5px;         /* Abgerundete Ecken */
    cursor: pointer;            /* Mauszeiger zeigt Hand */
}
            
/* Definiert eine Klasse "highlight" für hervorgehobene Texte */
.highlight {
    background-color: yellow;   /* Gelber Hintergrund */
    font-weight: bold;          /* Fettschrift */
}



ID-Selektoren

ID-Selektoren werden verwendet, um ein ganz bestimmtes HTML-Element zu gestalten, das eine eindeutige id-Eigenschaft besitzt. Im Gegensatz zu Klassen, die mehrfach auf einer Seite verwendet werden dürfen, darf jede id nur einmal vorkommen. In CSS setzt du ein # vor den ID-Namen – zum Beispiel #header. ID-Selektoren eignen sich besonders gut für zentrale Bereiche deiner Webseite wie Navigation, Hauptüberschriften oder spezielle Abschnitte, die sich klar vom Rest abheben sollen.

Beispiel


index.html


<h1 id="main-title">Willkommen auf meiner Website</h1>

style.css


#main-title {
    color: darkblue;
    font-size: 32px;
    text-align: center;
    margin-top: 40px;
}



Attributselektoren

Attributselektoren ermöglichen es dir, HTML-Elemente basierend auf ihren Attributen und Attributwerten zu stylen. Du kannst damit zum Beispiel gezielt alle Links mit einem bestimmten Ziel href, Eingabefelder mit einem bestimmten Typ (type) oder Elemente mit benutzerdefinierten Attributen ansprechen. Attributselektoren sind besonders nützlich, wenn Klassen oder IDs nicht ausreichen oder du selektiv auf bestimmte Eigenschaften von Elementen reagieren möchtest. Durch ihre Flexibilität helfen sie dir, deinen CSS-Code präziser und kontextabhängiger zu gestalten.

Beispiel


/* Wählt alle Links mit einem href-Attribut */
a[href] {
    color: blue;
    text-decoration: underline;
}

/* Wählt alle Links, die auf eine PDF-Datei verweisen */
a[href$=".pdf"] {
    color: red;
    font-weight: bold;
}

/* Wählt alle Eingabefelder mit dem Typ "text" */
input[type="text"] {
    border: 1px solid #ccc;
    padding: 5px;
}

/* Wählt alle Elemente mit einem data-role Attribut */
*[data-role] {
    background-color: #f0f0f0;
}



Pseudoklassen

Pseudoklassen sind Schlüsselwörter, die an Selektoren angehängt werden, um den Stil eines Elements in einem bestimmten Zustand zu definieren – ohne zusätzliche Klassen oder IDs im HTML-Code. Sie ermöglichen dir, auf Interaktionen wie Hover-Zustände oder die Position eines Elements im DOM zu reagieren. Beispiele sind :hover für das Verhalten beim Überfahren mit der Maus, :first-child für das erste Kind eines Elternelements oder :focus für ein aktives Eingabefeld. Pseudo-Klassen sind besonders hilfreich, um interaktives und dynamisches Styling umzusetzen – ganz ohne JavaScript.

Beispiel


index.html


<ul>
    <li>Erstes Element (wird hervorgehoben)</li>
    <li>Zweites Element</li>
</ul>
              
<a href="#">Fahr mit der Maus über mich!</a>
              
<br><br>
              
<input type="text" placeholder="Klick mich an">
              
<br><br>
              
<button>Drück mich!</button>

style.css


/* Wenn der Benutzer mit der Maus über den Link fährt */
a:hover {
    color: red;
    text-decoration: underline;
}

/* Das erste Kind eines Elements bekommt eine andere Hintergrundfarbe */
li:first-child {
    background-color: #f0f0f0;
}

/* Das Eingabefeld wird beim Fokus blau umrandet */
input:focus {
    border: 2px solid blue;
}

/* Elemente, die gerade aktiv (z. B. beim Klicken) sind */
button:active {
    background-color: #ccc;
}



Pseudoelemente

Pseudoelemente ermöglichen es dir, spezielle Teile eines Elements gezielt zu gestalten, ohne dafür zusätzliches HTML zu schreiben. Mit Pseudoelementen wie ::before und ::after kannst du Inhalte visuell vor oder nach einem Element einfügen, etwa für Icons, dekorative Linien oder animierte Effekte. Andere Pseudoelemente wie ::first-line oder ::selection helfen dir, gezielt einzelne Textbereiche zu stylen. Sie machen dein CSS flexibler, reduzieren den HTML-Aufwand und ermöglichen clevere visuelle Details.

Beispiel


index.html


<p class="tipp">
    Du kannst mit CSS Pseudoelementen Inhalte hinzufügen, ohne das HTML zu verändern.
</p>

style.css


.tipp::before {
    content: " ";
    color: orange;
    font-weight: bold;
}
                
.tipp::after {
    content: " ";
    color: gold;
}



content

Die Eigenschaft content wird hauptsächlich in Kombination mit Pseudoelementen wie ::before und ::after verwendet, um zusätzlichen Inhalt in ein Element einzufügen. Dabei kannst du Text, Symbole, Bilder oder sogar leere Strings einfügen, ohne das HTML anzupassen. content ist ein mächtiges Werkzeug, um Layouts und Designs zu verfeinern und kleine dekorative Details direkt per CSS zu steuern.

Beispiel


index.html


<h3>Wichtige Hinweise</h3>

style.css


h3::before {
    content: "[INFO] ";
    color: #2a7ae2;
    font-weight: bold;
    font-family: Arial, sans-serif;
}
            
h3::after {
    content: " — Ende der Information";
    font-style: italic;
    color: #555;
    margin-left: 5px;
}
            
h3 {
    margin-top: 30px;
}



padding

padding ist eine Eigenschaft, mit der du den inneren Abstand zwischen dem Inhalt eines Elements und seinem Rand (Border) festlegst. Dadurch kannst du bestimmen, wie viel Platz innerhalb eines Elements frei bleibt, damit Text oder andere Inhalte nicht direkt am Rand kleben. Padding sorgt also für mehr „Luft“ und verbessert die Lesbarkeit und das Design. Du kannst padding für alle vier Seiten (oben, rechts, unten, links) einzeln oder als Kurzschreibweise angeben.

Beispiel


index.html


<div class="box">
    Dieser Text hat unterschiedlich viel Abstand zum Rand oben, rechts, unten und links.
</div>

style.css


.box {
    background-color: #f0f0f0;
    border: 2px solid #333;
    padding: 20px 40px 10px 15px; /* oben rechts unten links */
    width: 300px;
    font-family: Arial, sans-serif;
}



border

Die Eigenschaft border definiert den sichtbaren Rahmen um ein Element. Du kannst damit die Stärke, Farbe und den Stil des Rahmens festlegen, z. B. durchgezogene, gestrichelte oder gepunktete Linien. Borders helfen dabei, Inhalte optisch abzugrenzen oder hervorzuheben. Du kannst einzelne Seiten (oben, rechts, unten, links) unterschiedlich gestalten oder mit der Kurzschreibweise alle Seiten gleichzeitig einstellen. Ausserdem lassen sich Ecken mit border-radius abrunden und Rahmen mit box-shadow optisch aufwerten.

Beispiel


index.html


<div class="kasten">
    <h2>Standardrahmen mit abgerundeten Ecken</h2>
    <p>Dieser Kasten hat einen blauen, durchgezogenen Rahmen mit abgerundeten Ecken und einem leichten Schatten. Er wirkt dadurch modern und hebt sich gut vom Hintergrund ab.</p>
</div>
            
<div class="kasten kasten-links">
    <h2>Individuelle Rahmenseiten</h2>
    <p>Hier sind die Rahmenseiten unterschiedlich gestaltet: Oben gestrichelt, rechts kein Rahmen, unten gepunktet und links ein dicker durchgezogener Rahmen. So kannst du gezielt Akzente setzen.</p>
</div>

style.css


.kasten {
    width: 350px;
    padding: 20px;
    margin: 20px auto;
    font-family: Arial, sans-serif;
    background-color: #fafafa;
                
    /* Rahmen komplett mit Kurzschreibweise */
    border: 4px solid #2a7ae2;
                
    /* Ecken abrunden */
    border-radius: 12px;
                
    /* Schatten für 3D-Effekt */
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}
            
/* Einzelne Rahmenseiten verschieden gestalten */
.kasten-links {
    border-top: 3px dashed #ff6f61;
    border-right: 0;
    border-bottom: 3px dotted #ff6f61;
    border-left: 5px solid #ff6f61;
    border-radius: 0;
    box-shadow: none;
    margin-top: 40px;
    padding: 15px;
    background-color: #fff0f0;
}



margin

margin ist eine Eigenschaft, mit der du den Aussenabstand eines Elements zu anderen Elementen festlegen kannst. Damit bestimmst du, wie viel Platz um ein Element herum frei bleibt, also der Abstand zum Nachbarinhalt oder zum Rand des Containers. margin kann für jede Seite (oben, rechts, unten, links) einzeln gesetzt werden, oder als Kurzschreibweise mit ein bis vier Werten. Mit margin: auto; kannst du Elemente auch horizontal zentrieren, wenn eine feste Breite definiert ist.

Beispiel


index.html


<div class="container">
    <div class="box">
        <p>Diese Box hat oben 10px, rechts 30px, unten 20px und links 5px Abstand zum nächsten Element.</p>
    </div>

    <div class="box2">
        <p>Diese Box ist horizontal zentriert dank <code>margin: auto;</code>.</p>
    </div>
</div>

style.css


.container {
    background-color: #e0f7fa;
    padding: 20px;
    width: 400px;
    font-family: Arial, sans-serif;
    margin: 30px auto;
    border: 2px solid #00796b;
}
            
.box {
    background-color: #4db6ac;
    color: white;
    padding: 15px;

    /* oben rechts unten links */
    margin: 10px 30px 20px 5px;

    border-radius: 8px;
}
            
.box2 {
    background-color: #00796b;

    /* zentriert horizontal */
    margin: auto;

    width: 150px;
    padding: 10px;
    border-radius: 8px;
    text-align: center;
    color: white;
}



fontfamily

Die CSS-Eigenschaft font-family bestimmt, welche Schriftart für Text in einem Element verwendet wird. Du kannst Systemschriftarten verwenden, Google Fonts einbinden oder eigene Schriftdateien im Webfont-Format (wie .ttf, .woff, .woff2, .otf) lokal hosten und importieren. Dafür nutzt du die @font-face-Regel, mit der du eine eigene Schriftdatei definierst und ihr einen Namen gibst, den du später in font-family verwenden kannst.

Browserschriftarten


p {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

Import über URL


<head>
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>

oder


@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
    font-family: 'Roboto', 'Helvetica Neue', Helvetica, Arial, sans-serif;
}

Lokal einbingen


@font-face {
    font-family: 'MeineCustomFont';
    src: url('fonts/MeineCustomFont.woff2') format('woff2'),
         url('fonts/MeineCustomFont.woff') format('woff'),
         url('fonts/MeineCustomFont.ttf') format('truetype');
    font-weight: normal;
    font-style: normal;
}
              
body {
    font-family: 'MeineCustomFont', Arial, sans-serif;
}



font-size

Die Eigenschaft font-size legt die Schriftgrösse von Text in einem HTML-Element fest. Du kannst die Grösse in verschiedenen Einheiten angeben, z. B. in px (Pixel), em (relativ zur Schriftgrösse des Elternelements), rem (relativ zur Wurzel-Schriftgrösse, meist html) oder %. Mit font-size kannst du die Lesbarkeit und die visuelle Hierarchie deiner Inhalte gezielt steuern. Für responsive Designs empfiehlt es sich, relative Einheiten wie em oder rem zu verwenden, da sie sich besser an unterschiedliche Bildschirmgrössen anpassen.

Beispiel


html {
    font-size: 16px;        /* Basisgrösse für rem */
}

h1 {
    font-size: 2rem;        /* entspricht 32px */
}

p {
    font-size: 1rem;        /* entspricht 16px */
}

.small-text {
    font-size: 0.875rem;    /* ca. 14px */
}

.large-text {
    font-size: 1.5em;       /* relativ zur Schriftgrösse des Elternelements */
}



font-style

Die Eigenschaft font-style wird verwendet, um die Schrift eines Textes schräg darzustellen. Typischerweise nutzt man sie für kursiven Text – also für Betonungen, Fachbegriffe oder Zitate. Die häufigsten Werte sind normal (Standardschrift), italic (echte Kursivschrift, wenn verfügbar) und oblique (künstlich geneigte Schrift). Während italic auf speziell gestaltete Zeichen zurückgreift, wird bei oblique die normale Schrift einfach geneigt – das wirkt oft weniger elegant. Du kannst font-style global im CSS oder gezielt für einzelne Elemente verwenden.

Beispiel


index.html


<p class="normal">Dies ist normaler Text.</p>
<p class="italic">Dies ist kursiver Text mit <code>italic</code>.</p>
<p class="oblique">Dies ist schräger Text mit <code>oblique</code>.</p>

style.css


.normal {
    font-style: normal;
}

.italic {
    font-style: italic;
}

.oblique {
    font-style: oblique;
}



font-weight

Die Eigenschaft font-weight steuert die Dicke bzw. Stärke der Schriftzeichen. Übliche Werte sind numerisch von 100 (sehr dünn) bis 900 (sehr fett), wobei 400 dem normalen Schriftgewicht entspricht und 700 für fett steht. Ausserdem kannst du die Schlüsselwörter normal und bold verwenden, die den Werten 400 bzw. 700 entsprechen. Nicht alle Schriftarten unterstützen alle Gewichtungen, daher solltest du prüfen, welche Schriftschnitte verfügbar sind – vor allem bei Webfonts. Mit font-weight kannst du Texte optisch hierarchisch strukturieren oder hervorheben.

Beispiel


index.html


<p class="normal">Dies ist normal gewichteter Text.</p>
<p class="bold">Dieser Text ist fett (bold).</p>
<p class="light">Dieser Text ist leicht (light).</p>
<p class="black">Dieser Text ist sehr fett (black).<p>

style.css


.normal {
    font-weight: normal; /* entspricht 400 */
}
              
.bold {
    font-weight: bold; /* entspricht 700 */
}
              
.light {
    font-weight: 300;
}
              
.black {
    font-weight: 900;
}



text-align

Die CSS-Eigenschaft text-align bestimmt, wie der Text innerhalb eines Elements horizontal ausgerichtet wird. Typische Werte sind left (links, Standard bei westlichen Sprachen), right (rechts), center (zentriert) und justify (Blocksatz). Mit text-align kannst du das Layout von Überschriften, Absätzen oder anderen Textblöcken steuern und so für bessere Lesbarkeit und Optik sorgen.
